实现无向图的深度优先遍历和广度优先遍历算法（图可能为非连通图）。
【输入】
  第1行：整数n（表示点数，默认取值是0-n-1） 整数m（图中的边数）（n小于500）  
  第2行：m条边的序列（s, e），s为边的起点编号，e为边的终点编号。如2 3表示V2到V3有边。
【输出】
  第1行，深度优先遍历结果
  第2行，广度优先遍历结果
  (DFS和BFS遍历过程中总是优先访问节点编号较小的点)

  例如，对于正方形有：
【输入】
  4 4//4个顶点，4条边
  0 1 1 2 2 3 3 0//0-1，1-2，2-3，3-0共四条边
【输出】
  0 1 2 3//DFS 
  0 1 3 2//BFS
【说明】
图的拓扑结构为：
0------------------3
|                  |
|                  |
|                  |
1------------------2
